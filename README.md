Schnellseite für Deine App

Wie lokal testen:

1. Öffne die Datei `quick_site/index.html` im Browser (Doppelklick) oder nutze einen Live-Server.
2. Kontaktformular ist nur UI‑Proof; Backend nicht angebunden.

Deployment (empfohlen):
- Netlify / Vercel: Repo pushen und Deploy über UI.
- GitHub Pages: Einfach branch gh-pages oder Action nutzen.

Änderungen: Passe Texte in `index.html`, Farben in `styles.css` (CSS-Variablen).
